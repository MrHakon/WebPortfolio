Portfolio

This is a couple of projects that I am currently working on. The languages used are mostly HTML, CSS and Java.

Template Page

When starting a new web project, there’s a lot of elements that are required for the project to function at all, and there’s a lot of elements that you’re probably going to use anyway. This template is a work in progress that should take care of the most essential stuff and provide a great starting point for a new project. In fact, this very website is based on this template.

Technologies: HTML and CSS (mostly)

check it out on codepen

Semantic Quiz

This program is written in Java, and uses the Apache Jena-library to query semantic data and turn it into a quiz. The query-part of the program is written in SPARQL, and is accessing the information from various online endpoints, such as DbPedia. There is currently no graphic user interface.

Technologies: Java, SPARQL, JSON, Apache Jena

Check it out on github

FreeCodeCamp

FreeCodeCamp is an online course that focuses on web development. This web portfolio is actually a part of the curriculum, and has to be approved in order for the course to be completed.

Technologies: HTML, CSS, JavaScript, Ajax, jQuery, MongoDB, and more

check out freecodecamp

FilmFacts

This is a website that presents films from a database (actually and object in its current state). It has search functionality, can display reviews, add reviews, and add films to lists.

Technologies: HTML, CSS, JavaScript

check it out on github
